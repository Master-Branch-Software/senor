## Design systems and principles

- [Refactoring UI (Adam Wathan, Steve Schoger)](https://www.refactoringui.com/) — practical visual design heuristics.
- [Material Design 3 (Google)](https://m3.material.io/) — token systems, type scale, color roles, dark mode.
- [Apple Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/) — platform conventions, motion, dark mode.
- [Atomic Design (Brad Frost)](https://atomicdesign.bradfrost.com/) — primitive/component/template hierarchy.
- [Anthony Hobday — Safe rules for visual design](https://anthonyhobday.com/sideprojects/saferules/) — defaults that rarely fail.

## Typography

- [Utopia (fluid type)](https://utopia.fyi/) — viewport-based type and space scales.
- [Type Scale](https://type-scale.com/) — modular scale calculator.
- [Modular Scale](https://www.modularscale.com/) — Tim Brown's original calculator.
- [Google Fonts](https://fonts.google.com/) — Latin and non-Latin coverage.
- [Fontsource](https://fontsource.org/) — self-hostable open fonts.
- [V-Fonts](https://v-fonts.com/) — variable fonts showcase.
- [Wakamai Fondue](https://wakamaifondue.com/) — font inspector for variable axes and OpenType features.

## Color and contrast

- [OKLCH color picker](https://oklch.com/) — perceptually uniform palette construction.
- [Radix UI Colors](https://www.radix-ui.com/colors) — token-shaped scales with built-in WCAG ratios.
- [Huemint](https://huemint.com/) — generative palette tooling.
- [Coolors](https://coolors.co/) — quick palette ideation.
- [Realtime Colors](https://realtimecolors.com/) — preview palette across a real layout.
- [Leonardo (Adobe)](https://leonardocolor.io/) — palette generation against contrast targets.
- [Contrast Tools](https://contrast.tools/) — pair-wise contrast inspector.
- [Colorable (Jxnblk)](https://colorable.jxnblk.com/) — pair grids for accessible combinations.
- [APCA contrast (Andrew Somers)](https://www.myndex.com/APCA/) — perceptual contrast model under consideration for WCAG 3.
- [Adrian Roselli — WCAG 3 contrast as of April 2026](https://adrianroselli.com/2026/04/wcag3-contrast-as-of-april-2026.html)
