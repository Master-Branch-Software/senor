# Inspiration Gallery

Curated, human-vetted directory of websites and design archives. Use these as a designer would: browse 5–10 examples in the relevant category to internalize structural patterns (hero composition, section pacing, type pairing, color temperament, motion vocabulary), then build from primitives in the brand's own voice. Do not copy a site as a template; absorb the _reasoning_ behind decisions that fit a similar brief.

## How agents should use this

- Open this file before generating a design from scratch when the brief mentions a feel, an industry, or a category. It is cheaper and produces better output than an open-web crawl.
- Pick 1–2 categories that match the brief's target feel and audience. Do not browse all categories.
- Visit 5–10 sites in those categories. Observe: hero archetype, section count and order, type stack, color move, motion intent, what is omitted.
- Note the _patterns_, not the pixels. "Brand X opens with a 90vh editorial hero, then jumps to a single product image at full bleed, then a 4-column case-study grid" — that is a reusable structure. The exact pixel and color choices belong to that brand.
- Audit the chosen reference against the brief: does the brand fit warrant this archetype? Does the audience read this way? If not, look further.
- After choosing a structural direction, build from primitives respecting all rules in chapters 03, 04, 05, 06.

Anti-patterns when using this list:

- Lifting layout, motion, and color wholesale from a single award-winning site. The site won because of _its_ brand fit; the next brand will not benefit from the same outfit.
- Browsing inspiration as a substitute for discovery. Inspiration follows the brief; it does not replace it.
- Generating sameness. If the first three references show centered hero + purple gradient + 3-feature grid, that is a signal to look further or design the opposite.
- Treating any site here as authoritative for accessibility or performance. Many awarded sites prioritize craft over Core Web Vitals or WCAG compliance — apply chapter rules first; treat these as visual prompts only.

## Categories

### Awards and craft showcases

For maximum-craft work, motion-heavy sites, agency portfolios. Helpful when the brief calls for "experiential," "creative," "premium agency," or "memorable."

- [Awwwards](https://www.awwwards.com/) — daily site of the day; agency and brand work.
- [FWA](https://thefwa.com/) — long-running showcase for cutting-edge interactive work.
- [CSS Design Awards](https://www.cssdesignawards.com/) — creative CSS-driven sites.
- [CSS Winner](https://www.csswinner.com/) — outstanding CSS-based designs.
- [CSS Nectar](https://cssnectar.com/) — gallery of creative CSS sites.
- [Hover States](https://hoverstat.es/) — interaction and microinteraction showcase.

### Landing pages (SaaS, marketing, conversion)

For product launches, SaaS marketing sites, lead-capture pages.

- [Landbook](https://land-book.com/) — large landing-page gallery.
- [Landingfolio](https://landingfolio.com/) — best-of landing pages with section views.
- [Landings](https://landings.design/) — innovative landing pages.
- [Lapa Ninja](https://www.lapa.ninja/) — landing-page inspiration.
- [Landing Picks](https://www.landingpicks.com/) — top landing-page designs.
- [One Page Love](https://onepagelove.com/) — single-page sites and templates.
- [Saaspo](https://saaspo.com/) — SaaS-focused landing pages, sectioned by type.

### Minimalist and editorial

For sites where typography, whitespace, and restraint carry the brand.

- [Minimal Gallery](https://minimal.gallery/) — minimalist site collection.
- [Unsection](https://unsection.com/) — minimalism-leaning inspiration plus 3,000+ section examples (hero, CTA, pricing).
- [Godly](https://godly.website/) — visually striking, typography-confident sites.

### Brutalism and anti-design

For raw, expressive, anti-corporate work; counter to AI defaults.

- [Brutalist Websites](https://brutalistwebsites.com/) — raw, intentionally rough designs.

### E-commerce

For product pages, Shopify-style stores, fashion, consumer goods.

- [Commerce Cream](https://commercecream.com/) — Shopify store inspiration.
- [Ecomm Design](https://ecomm.design/) — e-commerce design gallery.

### Curated visual feeds

For broad ambient inspiration; useful early in a project to absorb vocabulary.

- [Muzli](https://muz.li/) — curated daily design inspiration.
- [Maxi Best Of](https://maxibestof.one/) — top web design inspirations.
- [Site of Sites](https://siteofsites.com/) — aggregated quality websites.
- [Webdesign-Inspiration](https://www.webdesign-inspiration.com/) — web and mobile.
- [Wix Inspiration](https://www.wix.com/blog/topic/inspiration/) — Wix-built sites; useful as a control group.

### Portfolios

For agency, freelancer, designer-developer, and creative-personal sites.

- [Killer Portfolio](https://killerportfolio.com/) — portfolio inspiration.

### Designer communities (proceed with discernment)

For broader visual directions, illustration, and concept work. Many shots never ship; treat as mood, not as production reference.

- [Dribbble](https://dribbble.com/) — designer shots.
- [Behance](https://www.behance.net/) — long-form portfolios and case studies.
- [Pinterest](https://www.pinterest.com/) — broad visual mood boards.
- [Savee.it](https://savee.it/) — curated visual collections.
- [Cosmos](https://cosmos.so/) — curated boards across art and design (note: not the URL `cosmos.network`, which is a blockchain).
- [Designspiration](https://www.designspiration.com/) — visuals and layouts.
- [Instagram](https://www.instagram.com/) — designer accounts (search by name or hashtag).

### Templates (reverse-reference)

Look at templates to see what the _median_ solution is — then design _away_ from it.

- [Showit Marketplace](https://showit.co/marketplace/) — visual-builder templates.
- [Webflow Templates](https://webflow.com/templates) — Webflow templates.
- [Envato Elements](https://elements.envato.com/) — broad asset library.
- [Squarespace Templates](https://www.squarespace.com/templates) — Squarespace template gallery.

### Typography references

- [Fonts In Use](https://fontsinuse.com/) — font usage in the wild, indexed by type and industry.
- [Typographic Posters](https://typographicposters.com/) — poster typography for editorial inspiration.

### Branding and identity

- [Brand New](https://www.underconsideration.com/brandnew/) — branding and logo redesign coverage.

### Tutorials and pattern libraries

- [Codrops](https://tympanus.net/codrops/) — tutorials, pattern demos, motion explorations.
- [Abduzeedo](https://abduzeedo.com/) — design blog.

### Books and offline

- [Design book searches](https://www.amazon.com/s?k=design+books) — entry point for printed references; offline reading shapes taste in ways galleries cannot.

## Selection heuristics by feel

| Brief target feel                 | Start with                                      |
| --------------------------------- | ----------------------------------------------- |
| Premium, editorial, restrained    | Minimal Gallery, Godly, Fonts In Use, Brand New |
| Loud, expressive, creative agency | Awwwards, FWA, CSS Design Awards, Hover States  |
| Raw, anti-corporate, indie        | Brutalist Websites, Cosmos                      |
| SaaS, product, conversion-focused | Saaspo, Landbook, Landingfolio, Lapa Ninja      |
| E-commerce, retail, fashion       | Commerce Cream, Ecomm Design                    |
| Personal portfolio, agency        | Killer Portfolio, Behance                       |
| Typography-led                    | Fonts In Use, Typographic Posters, Godly        |
| Long-form storytelling            | Awwwards (case studies), Behance                |
| Pattern / section reference       | Unsection, Codrops                              |

## Maintenance

This list is human-curated and human-verified. URLs rot; before recommending a specific site to a user, do a quick check that the link resolves. When a domain dies or a gallery loses quality, remove it. New sources are welcome via the same review bar as chapter sources (see `AGENTS.md`).
