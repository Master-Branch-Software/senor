Non-negotiables, applied automatically

The skill silently enforces 10 baseline rules on every task — from image sizing to motion preferences — so you never ship broken defaults.

SKILL.md — excerpt 
- Every <img> has width, height, and meaningful alt.
- Likely LCP image: fetchpriority="high", never lazy-loaded.
- Body text ≥ 16 px, line-height 1.5, line length 60–75 ch.
- WCAG 2.2 AA contrast: 4.5:1 body, 3:1 large text / UI.
- Visible :focus-visible on every focusable element.
- prefers-reduced-motion honored on all motion.
- Native HTML first: <button>, <dialog>, <details>, popover.
- font-display: swap with matched-metric fallbacks.
- No positive tabindex. Two-space indent. kebab-case files.
- visible, not isVisible. Logical properties always.
