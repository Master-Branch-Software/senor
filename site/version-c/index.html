The problem

AI outputs the median.
The median is bad.

Every model trains on the same corpus: Tailwind tutorials, dribbble shots, SaaS landing-page boilerplate. The output is predictable. Indigo gradients. Div soup with no semantic structure. Copy full of "empower" and "streamline." Three feature cards. Always three.

It's not a hallucination problem. It's a training-data problem. The median web is mediocre, and AI produces the median with impressive consistency.

"The code we want to generate should look human, have good architectural and security decisions. It shouldn't create unnecessary abstractions — but find balance and create optimal solutions."

These guidelines are built from engineering blogs, official documentation, and professional codebases. Concrete rules. Not platitudes.

Clone the repository

Plain Markdown files. No install, no build step, no dependencies.

02

Add to Claude Code

Copy SKILL.md into your project's .claude/ folder. It loads on every task automatically.

No prompt engineering. No reminders. Claude reads the guidelines on every web task and follows the rules. That's it.
